## 15. 代码生成系统

框架基于数据库结构自动生成代码。

### 15.1 生成内容

| 类别 | 生成内容 | 目录 |
|------|----------|------|
| Table | 表映射类，字段常量 | runtime/Generate/Tables/ |
| TableDto | 数据传输对象 | runtime/Generate/TablesDto/ |
| Model | 业务模型类 | runtime/Generate/Models/ |
| CRUD | 基础 CRUD 服务 | runtime/Generate/Cruds/ |
| Admin | 后台管理控制器 | runtime/Generate/Admins/ |
| Proto | Protobuf 协议文件 | protos/ |
| API | 前端 API 代码 | 根据配置生成 |

### 15.2 字段冲突检测

框架会自动检测字段名冲突：

- **Table 类冲突**：字段名与 PHP 保留字冲突
- **DTO 类冲突**：字段名转换后重复

检测到冲突时，解析器会停止并提示修改建议。
