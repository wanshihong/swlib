## 18. 工具类

### 18.1 可用工具类

| 类 | 功能 |
|----|------|
| `Log` | 日志记录 |
| `File` | 文件操作 |
| `Str` / `StringConverter` | 字符串转换 |
| `Arr` | 数组操作 |
| `Url` | URL 处理 |
| `Http` | HTTP 请求 |
| `Ip` | IP 地址处理 |
| `Crypto` | 加密解密 |
| `Cookie` | Cookie 操作 |
| `Twig` | 模板引擎 |
| `Language` | 国际化 |
| `DevSslCert` | SSL 证书生成 |
