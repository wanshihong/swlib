## 8. 进程管理

框架支持自定义用户进程，适合后台数据处理、定时同步等场景。

### 8.1 创建自定义进程

```php
use Swlib\Process\Attribute\ProcessAttribute;

#[ProcessAttribute(
    interval: 60000,              // 执行间隔（毫秒）
    redirect_stdin_stdout: false,  // 是否重定向标准输入输出
    pipe_type: 0,                  // 管道类型
    enable_coroutine: true         // 是否启用协程
)]
class DataSyncProcess
{
    public function run($server, $process): void
    {
        // 每隔 interval 毫秒执行一次
        echo "开始数据同步..." . PHP_EOL;

        // 同步逻辑
        $this->syncData();
    }

    private function syncData(): void
    {
        // 数据同步逻辑
    }
}
```

### 8.2 进程配置

| 参数 | 类型 | 说明 |
|------|------|------|
| `interval` | int | 执行间隔（毫秒），0 表示只执行一次 |
| `redirect_stdin_stdout` | bool | 是否重定向标准输入输出 |
| `pipe_type` | int | 管道类型 |
| `enable_coroutine` | bool | 是否启用协程 |
